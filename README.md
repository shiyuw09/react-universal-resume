## âš™ï¸ Installation and Usage

### Prerequisites

- Node.js
- npm or yarn

### Steps

1. Navigate to the project's directory

   ```shell
   cd react-universal-resume
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

Visit `http://localhost:5173/` to view the app.
